## NextJS

### Prerequisites

Node version 20.12.x

### Cloning the repository

    git clone https://github.com/anilsoylu/content-converter.git

### Install packages

    npm i

### Start the app

    npm run dev

### Available commands

Running commands with npm `npm run [command]`

| command | description                              |
| ------- | ---------------------------------------- |
| `dev`   | Starts a development instance of the app |
|         |
